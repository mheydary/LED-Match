# LED-Match
A Raspberry PI Game

Author: Mohammadreza Hajy Heydary

Requirements:
  Raspberry PI
  Bread Board
  LEDs and Resistors
  Push Button Switches
  Breakout Board (Optional)
  
External Libraries: gpiozero

How to run:
  python3 LED_game.py
  
For bread boad setup guide, look at the attached pictures in the repository
